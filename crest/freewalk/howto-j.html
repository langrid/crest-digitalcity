実行方法

DS と CS の起動

FreeWalk を実行させるためには、まず空間管理サーバ(Directory Server : DS)と 空間サーバ(Communication Server : CS)を起動させる必要があります。 DS および CS は Windows、UNIX (Solaris で動作確認済) 両方で動きます。
※ nijo では DS(port 12345), CS(空間名 : FreeWalkSpace) が動いていますので、この操作は必要ありません。
Windows の場合

DS : Output\Release 内の FW_DS を実行して下さい。

CS : コマンドプロンプトを起動し、 ディレクトリ Output\Release 内に移動して次のコマンドを実行して下さい。

FW_CS [空間名] [DS が起動しているマシンの IP アドレス] [DS のポート番号(12345)]
(例 : FW_CS FreeWalkSpace 130.54.21.152 12345)
UNIX の場合

まず、ディレクトリ FW_DSCS と Comm を make が実行可能な場所にコピーして、 make を実行して下さい。fw_ds と fw_cs がコンパイルされます。その後、

DS : fw_ds を実行して下さい

CS : fw_cs を次の引数で実行して下さい。

fw_cs [空間名] [DS が起動しているマシンの IP アドレス] [DS のポート番号(12345)]
(例 : % fw_cs FreeWalkSpace 130.54.21.152 12345)

InternetExplorer での設定

FreeWalk は ActiveX を利用しており、 標準の設定のままではセキュリティ上実行ができません。 よって最初に InternetExplorer の設定をする必要があります。
  1. InternetExplorerを立ち上げる。
  2. 「ツール」メニューの「インターネットオプション」を開く
  3. 「セキュリティ」の中の「信頼済みのサイト」をクリックし、 その中の「サイト」を選ぶ。
  4. 「次の Web サイトをゾーンに追加する」の中に先ほど fwvr.ocx をコピーした ホストを入力し、「追加」を押す
    (二条ラボで起動する場合は下記の図のように入力して下さい)
  5. 「このゾーンのサイトにはすべてサーバの確認(https:)を必要とする」の チェックを外す。
  6. OK を押して設定を保存する。
これで FreeWalk を起動させる準備ができました。

FreeWalk の起動

起動する前に上記の設定をして、サーバを起動したことを確認して下さい。
  1. fwvrocx.html を開く (ここでもVer2.2が起動できます)
  2. 以下のダイアログボックスが現われるので、ユーザ名などを指定する。
  3. 「OK」を押せば FreeWalk が起動します。
    (サーバに接続できない場合は「CSに接続できません」というメッセージが出てきます)
