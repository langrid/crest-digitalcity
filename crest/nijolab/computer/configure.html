プリンタのインストール(Windows)

Last Update: Aug 24, 2000

1. Windows 95/98

○LaserShot LBP-730PSの場合

  1. 「コントロールパネル」→「プリンタ」→「プリンタの追加」
  2. 「ネットワークプリンタ」を選択．
  3. 「ネットワークパス」として「\\nijo\pr1」と入力．
  4. 製造元として「Canon」， プリンタとして「Canon LASER SHOT LBP-730PS 2F」を選択．

○LaserShot LBP-2040の場合（Color Printer）

このプリンタを使用するにはNetBEUIのプロトコルが必要です。
まずコントロールパネルのネットワークを開いてNetBEUIのプロトコルを サポートしているか確認してください。
NetBEUIをサポートしていない場合には以下の手順でインストールしてください。

  1. 「コントロールパネル」で「ネットワーク」を選択し、ネットワークの設定画面を表示
  2. 「追加」を選択
  3. 「プロトコル」を選択し、追加ボタンを押下
  4. 「製造元」から「Microfsoft」、「ネットワークプロトコル」から「NetBEUI」を選択し、OKボタンを押下
正常にインストールが終了すればネットワークコンピュータの中にLBP-2040が見えるようになっているはずです。

ドライバのインストール方法
ドライバは\\nijo\share\driver\printer\lbp2040\WIN9X 配下においてあります。
ドライバをインストール際複雑になるのでこれらを自分の端末にdownloadしておくことをおすすめします。
  1. 「コントロールパネル」で「プリンタ」を選択し、「プリンタの追加」を実行
  2. プリンタの接続で「ネットワークプリンタ」を選択
  3. 「ネットワーク パス」として「\\Lbp-2040\prn1」を入力
  4. プリンタ用のドライバが一覧に存在しないので「ディスク使用」を選択
  5. 「ディスクからインストール」でドライバの存在するdirectoryを入力し、OKを押下
  6. 必要であればプリンタ名を変更し、完了を選択

2. Windows NT4.0

○LaserShot LBP-730PS (pr1)の場合

  1. 「コントロールパネル」→「プリンタ」→「プリンタの追加」
  2. 「ネットワークプリンタサーバ」を選択．
  3. 「NIJO」上にあるインストールしたいプリンタを選択．
  4. 「NULLプリンタドライバーがない」と文句を言ってくるので「OK」を押す．
  5. 製造元として「Canon」， プリンタとして「Canon LASER SHOT LBP-730PS 2F」を選択．

NTP

Last Update: July 16, 2000

Network Time Protocol の略． NTPサーバに時刻を問い合わせることで， 計算機の内部時計を正確に保つことができます． Windows ではフリーソフトの「桜時計」が有名です．

NTPを利用するには，NTPクライアントの設定で， サーバを130.54.121.129にしてください． これ以外の上位（外部）のサーバを指定しないで下さい （ファイアウォールのため通信ができません）．
