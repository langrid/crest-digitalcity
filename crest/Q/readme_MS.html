Microsoft Agent - Q インタフェースの
起動方法

  • 起動方法

    実行環境 : IBM/PC互換機上で動作するWindows 2000，Windows 98
    (Windows NT4.0/XP/Me については未確認)
    Webにアクセス可能なネットワーク環境

    1. まず ここから Q処理系パック(zipファイル)をダウンロードする.

    2. ダウンロードしたQファイルを解凍する．

    3. 解凍したディレクトリにあるrunMSAgent.bat (もしくは /Program/MSAgent.exe )をダブルクリックすると，Microsoft Agentが起動し,音声エンジンで喋りながらシナリオの実行を始める．

      (MicrosoftAgentのキャラクターが登場しなかったり,喋らない時はこちら.)

      • 起動されるシナリオは,初期設定では/Data/scenario-msagent/ipc.qが実行される.
        /Property/MSAgent.ini の指定を変更することにより,他のシナリオも起動可能である.
        (詳しくは/document/MSAgent_ver1_read_me.docを参照のこと. )

    4. シナリオ実行後は,エージェントを右クリックし,メニューから「終了」を選び,終了確認で「はい」を選ぶと終了する．



    起動に関する更に詳しい情報は,/document 以下にあるMSAgent_ver1_read_me.docおよびgui-manual.pdfを御覧ください.

  • FAQ

    • MS Agentのキャラクターが登場しない,また喋らない.

      •

      Windows95などでは,Microsoft Agentが標準でインストールされません.
      (Windows98/2000/Me/XPでは標準でインストールされています.)
      また,音声合成エンジンは標準ではインストールされないため,キャラクターを音声合成で喋らせるためには音声合成エンジンをインストールする必要があります.
      以下の手順に従い,Microsoft Agentと音声合成エンジンをインストールしてください.

      1. Microsoft Agent本体 をダウンロードする．
        (Windows98/2000/Meでは必要なし)

      2. 使用するMicrosoft Agentのキャラクタをインストールする.
        ちなみにWindows 98/2000 では，標準でMerlin (魔法使い)が 入っている．
        必要であれば， オウムのPeedy をインストールすることもできる．また， Microsoftのサイト からRobby，Genieをダウンロードすることもできる.

      3. 以下の3つ実行ファイルをダウンロードし,音声合成エンジン(L&H社製，フリー版)をインストールする.

        • 音声合成エンジン(共通)
        • 日本語音声合成エンジン
        • 音声合成設定用コントロールパネル

      ただし，何らかの理由によりWebページにアクセスでき ない場合は， Microsoft社の公式サイト のページ左端にあるメニュー "Downloads" からそれぞれダウンロード可能です．
