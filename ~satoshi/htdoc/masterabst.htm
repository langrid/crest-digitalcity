1996年度修士論文概要

【論文題目】

生物的機能を持つ能動的人工視覚の構築
- 移動物体の検知・追跡及び形状識別 -

【論文概要】
視覚は生物の最も強力な感覚である．なぜならば，大量にある環境 情報を提供し，物理的に接触することなしに，環境と知的に相互作 用することを可能にしているからである．いうなれば，生物は視覚 を通して，物体の位置決定や同定，または移動物体を検知し，物体 間の関係を把握しているのである．近年，コンピュータビジョンの 研究において，このような生物の視覚を基にした，能動的視覚とい う概念が人間とロボットまたはロボットと環境とのインタラクショ ンを可能にするものとして，注目されている．

本研究では，複数の処理法を自ら最適に選択し，その結果から学習 していくような視覚も能動的視覚の概念の一つであると考え，この ような視覚系を生物的機能を持つ能動的人工視覚と定義する．した がって，その視覚系の構築を目指すことが研究目標である．生物， 特に人間は，環境の状況すなわち“動き”に応じて処理を分別する 能力を持っているといえる．このような処理分別能力は生物固有の 能力であり，本研究が目指す生物的機能の基本的なものの 1つと考 えられる．

ここで，生物的機能を持つ能動的人工視覚構築の第一段階として， この“動き”を物体の移動とし，その検知，追跡および形状識別処 理能力を有する処理分別化された視覚システムの構築を目的とする． これまでに，オプティカルフローやステレオ視を用いた移動物体の 検知処理の研究が幾つか報告されている．しかしながら，これらの 手法では移動物体の形状の識別問題が取り残されている．したがっ て，この問題を解決するために，本論文では形状識別処理に有効な 検知・追跡手法を提案することを中心課題とする．

処理分別機能を持つ視覚システムは，まずシステムが持つ複数の処 理プロセスが環境の状況に対応していなければならない．そこで， 環境の状況を記号化したパラメータが必要となる．たとえば，ステ レオ視を用いることにより，対象物体までの距離をパラメータとし て定義できる．しかし，本研究ではハードウェアの構成上，単眼視 システムとなるため，撮影される画像を 3つの領域に分割する．そ の領域によって対象物体までの距離を代表させる方式をとる．した がって，移動物体がどの領域に存在するかによって，処理を分別す ることになる．また，各状況に対応している処理目的を遠方にある 場合から順に以下に列挙する．

  • Level 0 : 移動物体の存在確認
  • Level 1 : 移動物体の移動量・方向検出
  • Level 2 : 移動物体の追跡・形状識別
このように，遠方に移動物体がある場合( 低位レベル ) には比較的簡単な処理を施し，逆に近方にある場合( 高位レベル ) は複雑な処理を行わせる．

実際に施す処理過程について順次簡単に説明する．まず低位レベル である Level 0 および 1では逐次撮影される輝度画像( 時系列画 像 )から差分画像を生成する． したがって，差分画像に画素群が 存在するならば，移動物体が存在することを意味する( Level 0 )． また，差分画像を水平・垂直方向の軸へ投影することにより，移動 物体が存在し得る領域を推定し，時系列画像と比較することにより， 移動物体の移動量・方向を大まかに算出する( Level 1 )．

そして，高位レベルに相当する Level 2では，本論文で提案するア ルゴリズムを用いて対象物体を追跡し，その結果を利用することに より，形状を識別する．一般に，物体の追跡アルゴリズムには高速 に演算を行わせるため，低解像度での相関演算を用いることが主流 である．しかしながら，その形状を識別するという観点では，必ず しもこの相関演算手法は有効であるとはいえない．そこで，本稿で は形状の特徴点でもある物体の頂点を追跡する点，すなわち注視点 として選択することにより，高解像度での追跡処理を可能にする． この注視点を抽出するために，2値画像の局所的特徴量である連結 数の概念を利用した拡張連結数を提案する．この拡張連結数は注目 画素に連結する連結成分数を表したものであり，この特性を利用す ることによって，頂点として抽出することができる．次に重要とな るのが追跡手法である．そこで，この抽出された画素群が形状の特 徴点として，さらには追跡する点としての能力を最大限発揮するた めに，一般的な画像処理手法である Convolution 処理を用いたア ルゴリズムを提案する．また，本手法によって，得られる追跡結果 を解析することにより，物体の形状を識別するアルゴリズムを確立 していく．

したがって，本論文は，処理分別機能を持つ視覚システムを構築す るとともに，新しい追跡アルゴリズムおよび形状識別アルゴリズム を提案し，それらを検証，考察するものである．
